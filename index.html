SIMON GAME

freeCodeCamp - Advanced Front End Development Project

Simon game shows a sequence of colors and sounds. Starting at one tone, player needs to follow the correct sequence shown by the computer. Every time the player enters the correct sequence, one more color is added until the total of 20 colors in sequence. In regular mode, every time that player enters the incorrect color the computer will show the correct sequence again. If strict mode is on, player won’t have a second chance and the game will restart if the wrong sequence is entered.

HTML5, CSS3 and jQuery

Tic Tac Toe

freeCodeCamp - Advanced Front End Development Project

Traditional Tic Tac Toe game. User has to choose between “X” and “O” and play first. Computer will make its move (at the moment only easy mode is available, computer will choose any empty space, making the game very easy). The first side to have a sequence of 3 values horizontally, vertically for diagonally will win the match. After every round the score will be updated.

HTML5, CSS3, Bootstrap and jQuery

JavaScript Calculator

freeCodeCamp - Advanced Front End Development Project

JavaScript calculator with addition, subtraction, multiplication and division operations. User has the choice to clear the last entered value or clear all the input field. The mathematical operations are chained together until user hit the equal button and the calculator will tell the correct output.

HTML5, CSS3, Bootstrap and jQuery

Pomodoro Clock

freeCodeCamp - Advanced Front End Development Project

The Pomodoro technique uses a timer to break down work into intervals, traditionally 25 minutes in length separated by 5 minutes’ breaks. This Pomodoro clock allows the user to set the session and break lengths. It starts with the session time, counting down until zero. When session time reaches 0, the break time starts. User can reset the clock at any time.

HTML5, CSS3, Bootstrap and jQuery

Wikipedia Viewer

freeCodeCamp - Intermediate Front End Development Project

This app uses Wikipedia’s API. User is able to click the image and see a random Wikipedia entry. As well, user can search for any entry by using the search Box. The results will be shown on the bottom of the page.

HTML5, CSS3, Bootstrap, jQuery, Ajax and JSON

Twitch TV streamers

freeCodeCamp - Intermediate Front End Development Project

This project uses Twitch TV API, showing a list of streamers, their name, channel logo and status. If the Twitch user is currently streaming it is possible to see additional information regarding what they are streaming. It is possible to filter the page by All users, Online users and Offline users.

HTML5, CSS3, Bootstrap, jQuery, Ajax and JSON

Local Weather

freeCodeCamp - Intermediate Front End Development Project

This project uses 2 APIs, the first one gets user’s location through IP and the second one shows the weather forecast for that specific location. Users can check the weather condition, temperature, humidity, pressure and wind speed. There is an option to change from Metric to Imperial units.

HTML5, CSS3, Bootstrap, jQuery, Ajax and JSON

Quote Machine

freeCodeCamp - Intermediate Front End Development Project

This is a random quote machine. By using an API, user will receive a new quote every time the “New Quote” button is pressed. Besides getting a new quote, this button will trigger a background color change as well. There is the option to share the quote on user’s Twitter by clicking the “Tweet” button.

HTML5, CSS3, Bootstrap, jQuery, Ajax and JSON
